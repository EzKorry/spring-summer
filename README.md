## 사전 설치

아래 프로그램은 컴퓨터에 설치해야 실행이 됩니다. 

- `Apache Maven` : 3.6.3
- `JDK` : java version "14.0.1" 2020-04-14

## 실행법

1. 코드 실행

  ```bash
  cd project_path
  mvn spring-boot:run
  ```

1. 인터넷 브라우저 접속 후 `localhost:8080` 주소 입력
